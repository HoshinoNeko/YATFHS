API Reference

▶

Upload a file

# Basic upload
curl -F "file=@/path/to/file.zip" \
     "/api/v1/upload"

# With custom expiry (seconds)
curl -F "file=@/path/to/file.zip" \
     -F "expire=14400" \
     "/api/v1/upload"

Response

{
  "status": "success",
  "data": {
    "url": "https://your-host/{id}/{filename}"
  }
}

Parameters

file    [required]  multipart file, max 100 MB
expire  [optional]  seconds until deletion, default 3600

Download

GET /{id}/{filename}
YATFHS files auto-deleted on expiry · no registration required
